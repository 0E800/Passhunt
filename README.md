# Passhunt
Passhunt is a simple tool for searching of default credentials for network devices, web applications and more.

# How to install?
<pre>
git clone https://github.com/Viralmaniar/Passhunt.git
cd Passhunt
pip3 install -r requirements.txt
python Passhunt.py
</pre>
